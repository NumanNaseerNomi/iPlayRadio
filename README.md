# 📻 iPlayRadio - Worldwide Radio Channels

A modern Nuxt.js application for discovering and listening to radio channels from around the world. iPlayRadio provides a seamless interface to search and explore thousands of radio stations based on various criteria.

## 🌐 Live Demo

Check out the live application: [iPlayRadio](https://numannaseernomi.github.io/iPlayRadio/)

## ✨ Features

- **Global Radio Access**: Listen to radio channels from countries worldwide
- **Advanced Search**: Find stations by:
  - Country
  - Country Code
  - Language
  - Station Name
  - State/Region
  - Tags/Categories
- **Smart Suggestions**: Get recommended channels
- **Tag Exploration**: Browse stations through an organized tag list
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Theme Support**: Auto, Dark, and Light mode for comfortable viewing
- **Real-time Loading**: Smooth content loading with loading states

## 🛠️ Built With

- **Nuxt.js** - Vue.js framework for server-side rendering
- **Vue.js 3** - Progressive JavaScript framework
- **Radio Browser API** - Free radio station database
- **Bootstrap 5** - For styling and responsive design
- **Pinia** - State management
